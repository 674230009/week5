# 🍳 Week 5:ระบบดึงข้อมูลเมนูอาหารด้วย PHP OOP

โปรเจกต์นี้เป็นระบบดึงข้อมูลเมนูอาหารและวัตถุดิบจากฐานข้อมูล MySQL มาแสดงผลบนหน้าเว็บในรูปแบบของการ์ด (Food Cards) โดยใช้หลักการเขียนโปรแกรมเชิงวัตถุ (OOP) และเชื่อมต่อฐานข้อมูลอย่างปลอดภัยด้วย PDO

---

## 📂 โครงสร้างไฟล์ในโปรเจกต์

*   `db.php` : ไฟล์เชื่อมต่อฐานข้อมูล MySQL ด้วย PDO พร้อมระบบดักจับ Error และป้องกัน SQL Injection
*   `food.php` : คลาส `Food` (OOP) สำหรับจัดการข้อมูลเมนูอาหาร ประเภท และเก็บอาร์เรย์ของวัตถุดิบ
*   `recipe.php` : คลาส `Recipe` (OOP) สำหรับจัดการข้อมูลวัตถุดิบแต่ละชนิด (ชื่อ, ปริมาณ, หน่วย)
*   `pd.php` : หน้าหลักที่ใช้ดึงข้อมูลดิกจากฐานข้อมูล แปลงเป็น Object และประมวลผลออกมาเป็นหน้าเว็บ (HTML/CSS Grid)

---

## 🛠️ รายละเอียดคลาสและฟังก์ชันหลัก

### 1. คลาส Recipe (`recipe.php`)
ทำหน้าที่รับข้อมูลวัตถุดิบรายชิ้น ประกอบไปด้วย:
*   `$name` (ชื่อวัตถุดิบ), `$valor` (ปริมาณ), `$unit` (หน่วย)
*   `get_details()` : คืนค่าเป็น String ที่จัดฟอร์แมตวัตถุดิบเรียบร้อยแล้ว

### 2. คลาส Food (`food.php`)
ทำหน้าที่รวมข้อมูลอาหารและการ์ดแสดงผล ประกอบไปด้วย:
*   `$name` (ชื่ออาหาร), `$type` (ประเภทอาหาร), `$recipe` (อาร์เรย์ที่เก็บ Object ของคลาส Recipe)
*   `add_recipe(Recipe $ing)` : ฟังก์ชันสำหรับเพิ่มวัตถุดิบ (บังคับว่าต้องเป็น Type Recipe เท่านั้น)
*   `get_details()` : คืนค่าเป็นโครงสร้าง HTML ของการ์ดอาหารพร้อมวนลูปแสดงวัตถุดิบด้านใน

---

## 💻 วิธีการติดตั้งและรันโปรเจกต์

1. นำโฟลเดอร์ `week5` ไปวางไว้ใน `C:/xampp/htdocs/`
2. สร้างฐานข้อมูลใน phpMyAdmin ชื่อ `week5`
3. สร้างตารางข้อมูลให้สอดคล้องกับระบบ:
    *   ตาราง `foods` (ประกอบด้วย id, name, type)
    *   ตาราง `recipes` (ประกอบด้วย id, food_id, name, valor, unit)
4. เปิดเบราว์เซอร์แล้วไปที่ ลิงก์: `http://localhost/week5/pd.php`
